# Projet OZformation - Plateforme de Formation IA

Ce projet vise à créer une plateforme de formation en ligne complète sur l'Intelligence Artificielle, spécifiquement conçue pour les auto-entrepreneurs. La plateforme inclura des modules de formation structurés, des vidéos, des quiz interactifs (mini-jeux Pac-Man), un système d'inscription et de suivi des apprenants, une interface d'administration sécurisée, et la génération automatique d'attestations de réussite.

## Objectifs Clés

*   **Site web autonome** : Hébergé sur le nom de domaine OZformation.
*   **Public Cible** : Auto-entrepreneurs.
*   **Contenu** : Formation organisée en blocs thématiques avec vidéos et contenu textuel.
*   **Interactivité** : Quiz sous forme de mini-jeux Pac-Man IA.
*   **Fonctionnalités Utilisateurs** : Enregistrement, codes d'accès, suivi de progression, emails automatiques, attestation de réussite.
*   **Interface d'Administration** : Gestion des accès, apprenants, suivi des scores.
*   **Hébergement** : Sécurisé et géré par Manus, site responsive.

## Structure de la Formation

*   **Bloc 0** : Introduction et orientation (vidéos, quiz diagnostic, page HTML imprimable).
*   **Bloc 1** : Fondamentaux de l'IA (vidéos, propriété intellectuelle, RGPD, quiz Pac-Man).
*   **Bloc 2** : Maîtriser les prompts (vidéo, quiz Pac-Man).
*   **Bloc 3** : Applications pratiques par médias (texte, images, musique, vidéo - vidéos, quiz Pac-Man).
*   **Bloc 4** : Business et stratégie (modules spécialisés, quiz final global).

## Technologies Proposées

*   **Frontend** : HTML5, CSS3, JavaScript (avec React pour les composants interactifs et la gestion d'état).
*   **Backend** : Python avec Flask pour l'API RESTful et la logique métier.
*   **Base de Données** : SQLite pour la gestion des utilisateurs, des scores, des codes d'accès et des données de progression.
*   **Hébergement** : Serveur Linux géré par Manus.
